Deploying Basic Latin Encoding

Displaying icon fonts

If an icon font is created using Basic Latin encoding, icons are used in place of the usual letterforms like “l” and “a”. While not as common as icon fonts that use PUA or symbol encoding, there are multiple icon fonts that use this approach.

To display an icon using this encoding, you can simply type the letter or character of the icon you need, and assign the icon font to the character through the use of the font-family property. While simple to use, the downside of this technique is that screen readers will still read the character and if the icon font fails to load you’re left with an unnecessary character on the page rendered in the default fallback font.

Aligning text and icons

When icon fonts are created each icon is drawn inside of a glyph’s Em box. This box controls the width and height of letterforms as well as where the character sits on the font’s baseline. This means that for icon fonts how the designer positions and sizes icons within its Em box has a direct effect on how the icon will align with text. In some cases proper alignment won't require any adjustment at all, while in others you may need to modify font size, vertical margins, or vertical alignment to get the proper relationship between icon and text. It's worth noting as well that you should expect variations within each icon font, so rarely will a global alignment class work for the entire icon font.

Normalizing icon fonts

Ensure that your icon fonts look consistent

When using icon fonts it’s tempting to simply set the font family declaration to display your icon font and ignore the element’s overall font properties. It's important to remember that icon fonts are still fonts and properties like font-weight font-style, text-transform, and others can have a negative effect on how the icons display. Because of this it's a good idea to normalize those properties through either a single dedicated selector or each time you assign the icon’s font-family.

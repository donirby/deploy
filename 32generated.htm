Using generated content

A better way to display icons

To avoid having characters or html entities directly in your HTML, you can use generated content to display icons. The ::before and ::after pseudo-selectors allow us to insert content before or after the content of an element. By choosing icon fonts as the content to display we keep our markup free of extraneous code and we start down the path of making our icons more friendly to accessibility devices.

Here's how it works.


.icon:before {
	font-family: 'my icon font';
	content: "\e000";
	font-size: 1em;
}

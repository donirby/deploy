Using CSS Animations with icon fonts

Styling for icon fonts can extend well beyond basic text formatting. You can change the appearance of icons by using CSS transforms, transitions, and even animations. In many cases icon fonts will include icons that with animation in mind. Most icon font sets, for example will contain some type of spinner or loader icon designed to indicate loading or some other type of processing.
